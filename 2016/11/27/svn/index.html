windows SVN服务器搭建和使用（一）

发表于 2016-11-27   |     |  

安装软件准备

TortoiseSVN_1.9.4.27285_x64.msi（SVN客户端）
LanguagePack_1.9.4.27285-x64-zh_CN.msi（SVN客户端汉化包）
VisualSVN-Server-3.5.6-x64.msi（SVN版本控制）

博主百度云盘分享:http://pan.baidu.com/s/1c2Fr1Oc（64位）
:
或者按照以下步骤里面提供的官网网址进行下载。

下载

64位系统的可以直接在我分享的百度云盘下载快点，如果是32位就按照我以下的操作就可以了。

Subversion下载

现在已经迁移到 apache网站了。下载地址
http://subversion.apache.org/packages.html
进入网页拖到最底部就能看到
windows操作系统下面的二进制文件包一共有5种，如图
:
个人认为最好用VisualSVN server 服务端和 TortoiseSVN客户端搭配使用.
点开VisualSVN ，国外网站会有点卡，稍等片刻呃。
